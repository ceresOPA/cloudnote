## 一、概述

### OSI模型与TCP/IP模型

| 模型               | 分层                                                       |
| ------------------ | ---------------------------------------------------------- |
| OSI模型（七层）    | 物理层、数据链路层、网络层、传输层、会话层、表示层、应用层 |
| TCP/IP模型（四层） | 网络接口层、网际层、运输层、应用层                         |
